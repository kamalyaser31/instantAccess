NVDA instantAccess 插件
=======================

*   **作者：** Kamal Yaser
*   **版本：** 2026.4
*   **兼容性：** NVDA 2024.1 及更高版本

简介
--------

**instantAccess** 是一款 NVDA 效率插件，可用于创建自定义快捷键层。无需记忆大量复杂快捷键：先按快捷键进入“instantAccess 模式”，再按一个键，即可启动应用程序、执行工作流、插入文本等。

它不只是一个启动器，还能将多个**操作**按顺序组合到一个**项目**中，让一个快捷键完成整套流程。

主要功能
------------

*   **多操作项目：** 为一个快捷键设置一组按顺序执行的操作。
*   **延迟控制：** 可为每项操作设置执行前等待时间，并统一设置操作间隔。
*   **应用程序专用快捷键：** 快捷键既可全局生效，也可限定在特定应用程序中。
*   **丰富的操作类型：**
    *   启动网站、程序、文件夹和文件。
    *   从可搜索的列表中执行 NVDA 命令。
    *   插入、复制或粘贴可重复使用的文本片段。
    *   模拟按键序列，并可调节按键间隔。
*   **集中管理：** 功能完善的设置面板，用于添加、编辑、测试和整理项目。
*   **导入与导出：** 以 JSON 格式备份和分享您的配置。
*   **可调节的详细程度：** 可选择详细反馈（初级）或简短的提示音（高级）。

内置快捷键
------------------

*   `NVDA+E`：开启或关闭 instantAccess 模式。
*   `NVDA+Shift+E`：读出当前焦点所在应用程序的名称，便于创建应用程序专用快捷键。
*   `NVDA+Shift+E`（快速连按两次）：将应用程序名称复制到剪贴板。
*   `Esc`（instantAccess 模式开启时）：退出该模式且不运行任何命令。

配置
-------------

您可以在 NVDA 设置面板中配置此插件：

1.  打开 **NVDA 菜单** (`NVDA+N`)。
2.  依次选择 **选项** > **设置**。
3.  选择 **instantAccess** 分类。

主设置面板列出了所有已配置的项目，显示它们的名称、类型、快捷键及操作摘要。在此处，您可以**添加**、**编辑**、**删除**和**测试**您的项目。

理解“项目”与“操作”
-------------------------------

该插件包含两个核心概念：**项目（Items）**和**操作（Actions）**。

*   **项目** 是您创建的基本单元，包含名称、快捷键以及一个或多个要执行的操作；还可限定在特定应用程序中生效，并设置操作间隔。
*   **操作** 是项目中的单一步骤。当您触发一个项目时，它会按顺序执行其中的操作。

### 创建和编辑项目

当您添加或编辑项目时，会弹出一个包含以下选项的对话框：

*   **名称：** 便于识别且不与其他项目重复的名称。
*   **操作列表：** 该项目要执行的所有操作。您可以**添加**、**编辑**、**删除**、**上移**和**下移**操作以调整执行顺序。
*   **操作间隔（秒）：** 每项操作执行完毕后等待的时长（最后一项除外）。
*   **仅在指定应用程序中启用此快捷键：** 勾选后，该快捷键仅在指定应用程序中生效。
    *   **应用程序名称：** 启用限制后，必须填写准确的应用程序名称（可用 `NVDA+Shift+E` 获取）。
*   **快捷键：** 用于录入该项目快捷键的按钮。

### 操作类型

添加或编辑操作时，您可以从以下几种类型中选择：

*   **网站：** 在默认浏览器中打开网址。
*   **程序：** 启动可执行文件 (`.exe`)。您还可以提供可选的**参数**。
*   **文件夹：** 在文件资源管理器中打开文件夹。
*   **文件：** 使用系统默认的关联应用程序打开任意文件。
*   **NVDA 命令：** 执行 NVDA 脚本。您可以在对话框中搜索并选择所有可用的 NVDA 及插件命令。
*   **文本片段：** 插入可重复使用的文本。此操作有三种模式：
    *   **输入：** 模拟输入文本。您可以设置**字符输入间隔**（秒）。
    *   **复制：** 将文本放入剪贴板。
    *   **粘贴：** 将文本放入剪贴板，再模拟按下 `Ctrl+V` 粘贴文本。
*   **按键序列：** 模拟一系列按键或快捷键组合（每行一个）。
    *   **语法：** 输入按键或快捷键名称（例如，`ctrl+alt+del` 或 `shift+f10`）。
    *   **重复：** 在按键名称后添加空格和次数即可重复按键（例如，`down 5` 表示按 5 次 `下光标`）。
    *   **按键间隔：** 设置每次按键之间的暂停时间（秒）。
*   **执行此操作前等待（秒）：** 执行该操作前等待的时长。

应用程序专用快捷键
----------------------

您可以让同一快捷键在不同应用程序中执行不同操作。

*   **全局快捷键**不受应用程序限制，可在任何位置使用。
*   **应用程序专用快捷键**仅在目标应用程序获得焦点时生效。

创建应用程序专用快捷键的步骤：

1.  在项目对话框中勾选“仅在指定应用程序中启用此快捷键”复选框。
2.  切换到目标应用程序。
3.  快速按两次 `NVDA+Shift+E` 将其名称复制到剪贴板。
4.  将名称粘贴到“应用程序名称”字段中。

如果全局快捷键和应用程序专用快捷键使用相同按键，则目标应用程序获得焦点时，**应用程序专用快捷键始终优先**。

详细程度
----------------

您可以在主设置面板中调整插件的反馈详细程度：

*   **初级（默认）：** 提供完整的语音提示（例如，“instantAccess 已开启，已加载 10 个命令。”）。
*   **高级：** 使用简短的提示音进行反馈（例如，开启或关闭时播放一声提示音）。

导入与导出
-----------------

您可以使用**导入设置**和**导出设置**按钮备份完整配置，或将配置迁移到另一台计算机。配置保存在 `config.json` 文件中。

**注意：** 导入配置将覆盖您现有的设置。
