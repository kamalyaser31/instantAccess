NVDA instantAccess 插件
=======================

*   **作者：** Kamal Yaser
*   **版本：** 2026.2
*   **兼容性：** NVDA 2019.3 及更高版本

简介
--------

**instantAccess** 是一个功能强大的 NVDA 生产力插件，允许您创建自定义快捷键层。您无需记住数十个复杂的快捷键，只需按下一个按键激活“instantAccess 模式”，然后按下简单的按键操作即可启动应用程序、运行小型工作流、插入文本等等。

该插件不仅仅是一个简单的启动器，它允许您将多个**操作**链式组合成一个序列化的**项目**，从而将单一的快捷键变成强大的宏。

主要功能
------------

*   **多操作项目：** 为单一快捷键分配一系列操作。
*   **时机控制：** 为每个操作设置自定义延迟，并为项目中的所有操作设置共享间隔。
*   **程序特定快捷键：** 可以将快捷键设置为全局有效，也可以限制在特定程序中。
*   **丰富的操作类型：**
    *   启动网站、程序、文件夹和文件。
    *   从可过滤的列表中执行 NVDA 命令。
    *   插入、复制或粘贴可重复使用的文本片段。
*   **集中管理：** 详尽的设置面板，用于添加、编辑、测试和整理您的项目。
*   **导入与导出：** 以 JSON 格式备份和分享您的配置。
*   **可调节的详细程度：** 在详细反馈（初级）或简洁音效（高级）之间进行选择。

内置快捷键
------------------

*   `NVDA+E`：开启或关闭 instantAccess 模式。
*   `NVDA+Shift+E`：读出当前焦点程序的名称。这对于创建程序特定快捷键非常有用。
*   `NVDA+Shift+E`（快速连按两次）：将程序名称复制到剪贴板。
*   `Escape`（在 instantAccess 模式激活时）：直接退出该模式而不运行任何命令。

配置
-------------

您可以在 NVDA 设置面板中配置此插件：

1.  打开 **NVDA 菜单** (`NVDA+N`)。
2.  进入 **选项** -> **设置**。
3.  选择 **instantAccess** 分类。

主设置面板列出了所有已配置的项目，显示它们的名称、类型、快捷键及操作摘要。在此处，您可以**添加**、**编辑**、**删除**和**测试**您的项目。

理解“项目”与“操作”
-------------------------------

该插件围绕两个核心概念构建：**项目（Items）** 和 **操作（Actions）**。

*   **项目** 是您创建的基本单元。它具有名称、快捷键（按键操作）以及要执行的一个或多个操作列表。它还可以限制在特定程序中，并在操作之间设置暂停间隔。
*   **操作** 是项目中的单一步骤。当您触发一个项目时，它会按顺序执行其中的操作。

### 创建和编辑项目

当您添加或编辑项目时，会弹出一个包含以下选项的对话框：

*   **名称：** 项目的唯一描述性名称。
*   **操作列表：** 该项目将执行的所有操作列表。您可以**添加**、**编辑**、**删除**、**上移**和**下移**操作以定义执行顺序。
*   **操作之间的间隔（秒）：** 在序列中的每个操作（最后一个除外）*之后*发生的暂停（以秒为单位）。
*   **将此快捷键限制在...：** 一个复选框，用于使快捷键特定于程序。
    *   **程序名称：** 如果勾选了限制，您必须提供准确的程序名称（使用 `NVDA+Shift+E` 获取）。
*   **快捷键：** 用于捕获该项目按键组合的按钮。

### 操作类型

添加或编辑操作时，您可以从以下几种类型中选择：

*   **网站：** 在默认浏览器中打开 URL。
*   **程序：** 启动可执行文件 (`.exe`)。您还可以提供可选的**参数**。
*   **文件夹：** 在文件资源管理器中打开文件夹。
*   **文件：** 使用其默认关联的应用程序打开任何文件。
*   **NVDA 命令：** 执行 NVDA 脚本。会弹出一个对话框，允许您过滤并从所有可用的 NVDA 和插件命令中进行选择。
*   **文本片段：** 插入可重用的文本。此操作有三种模式：
    *   **键入：** 模拟键入文本。您可以设置字符之间的自定义**键入延迟**（以秒为单位）。
    *   **复制：** 将文本放入剪贴板。
    *   **粘贴：** 将文本放入剪贴板，然后模拟 `Ctrl+V` 进行粘贴。
*   **执行此操作前的延迟：** 在此特定操作运行*之前*发生的暂停（以秒为单位）。

程序特定快捷键
----------------------

您可以让快捷键根据您所处的程序表现出不同的行为。

*   **全局快捷键**没有程序限制，在任何地方都有效。
*   **程序特定快捷键**仅在目标程序处于焦点时才激活。

创建程序特定快捷键的步骤：

1.  在项目对话框中勾选“将此快捷键限制在...”复选框。
2.  切换到目标程序。
3.  快速按两次 `NVDA+Shift+E` 将其名称复制到剪贴板。
4.  将名称粘贴到“程序名称”字段中。

如果全局快捷键 and 程序特定快捷键使用相同的按键操作，则当目标程序激活时，**程序特定快捷键将始终优先**。

详细程度级别
----------------

您可以在主设置面板中更改插件为您提供的反馈量：

*   **初级（默认）：** 提供完整的语音消息（例如，“instantAccess 已开启。已加载 10 个命令。”）。
*   **高级：** 使用简短的声音和音调以提高速度和效率（例如，开启/关闭时发出“哔”的一声）。

导入与导出
-----------------

您可以使用**导入设置**和**导出设置**按钮备份您的整个配置或将其移动到另一台计算机。配置存储在 `config.json` 文件中。  
**注意：** 导入配置将覆盖您现有的设置。
